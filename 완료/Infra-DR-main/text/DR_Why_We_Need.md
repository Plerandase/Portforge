# DR(재해 복구) 구성 이유

## 1. 비즈니스 관점

### 서비스 연속성
- **단일 리전 장애 시 서비스 완전 중단**
  - 2023년 AWS ap-northeast-2 장애: 약 4시간 서비스 중단 사례 다수
  - 2021년 AWS us-east-1 장애: 전 세계 수많은 서비스 영향
- **팀 협업 플랫폼 특성**: 업무 시간 중 서비스 중단은 사용자 생산성에 직접적 영향

### 사용자 신뢰도
- 서비스 중단 → 사용자 이탈 → 경쟁 서비스로 전환
- 한 번의 대규모 장애가 브랜드 이미지에 장기적 영향

### 데이터 보호
- 프로젝트 데이터, 팀 정보, 채팅 기록 등 사용자 핵심 자산
- 데이터 손실 시 복구 불가능 → 서비스 신뢰도 치명적 손상

## 2. 기술적 관점

### 단일 리전의 취약점
```
현재 구성 (Seoul Only):
┌─────────────────────────────────────┐
│         ap-northeast-2 (Seoul)      │
│  ┌─────┐  ┌─────┐  ┌─────┐         │
│  │ EKS │  │ RDS │  │ S3  │         │
│  └─────┘  └─────┘  └─────┘         │
│         Single Point of Failure     │
└─────────────────────────────────────┘
         ↓ 리전 장애 시
      서비스 완전 중단
```

### DR 구성 후
```
┌─────────────────────┐     ┌─────────────────────┐
│  ap-northeast-2     │     │  ap-northeast-1     │
│     (Seoul)         │────▶│     (Tokyo)         │
│    [Primary]        │복제 │      [DR]           │
└─────────────────────┘     └─────────────────────┘
         ↓ 리전 장애 시
   Tokyo로 자동 Failover
     서비스 계속 운영
```

### AWS 리전 장애 통계
- AWS 리전 장애 발생 빈도: 연 1~2회 (부분 장애 포함)
- 평균 복구 시간: 1~4시간
- DR 없이 대기 시 비용: 시간당 매출 손실 + 사용자 이탈

## 3. 비용 대비 효과

### 투자 비용
| 항목 | 월 비용 |
|------|---------|
| Primary (Seoul) | ~$230 |
| DR 추가 (Tokyo) | ~$204 |
| **총 비용** | **~$434** |

### 얻는 가치
| 지표 | DR 없음 | DR 있음 |
|------|---------|---------|
| 가용성 | 99.9% | 99.99% |
| 연간 다운타임 | 8.76시간 | 52분 |
| RTO | 무한 (AWS 복구 대기) | 30분 이내 |
| RPO | 백업 주기 의존 | 5초 이내 |

### ROI 계산
```
월 $204 투자로:
- 연간 다운타임 8시간 → 1시간 감소
- 시간당 기회비용 $100 가정 시: 연 $700 절감
- 사용자 신뢰도 및 브랜드 가치: 측정 불가 (Priceless)
```

## 4. 규정 준수 및 SLA

### 엔터프라이즈 고객 요구사항
- 대기업/공공기관은 DR 구성을 계약 조건으로 요구
- ISO 27001, SOC 2 등 보안 인증에서 DR 계획 필수

### SLA 보장
- DR 없이 99.99% SLA 보장 불가능
- 고객에게 명확한 RTO/RPO 제시 가능

---

## 5. 결론: DR이 필요한 핵심 이유

| 이유 | 설명 |
|------|------|
| **1. 서비스 연속성** | 리전 장애 시에도 서비스 유지 |
| **2. 데이터 보호** | 핵심 데이터 실시간 복제로 손실 방지 |
| **3. 사용자 신뢰** | 안정적인 서비스로 사용자 유지 |
| **4. 비용 효율** | 월 $204로 가용성 10배 향상 |
| **5. 경쟁력 확보** | DR 구성은 성숙한 서비스의 기본 요건 |

**"DR은 보험과 같다. 필요할 때 없으면 치명적이고, 있으면 안심할 수 있다."**
